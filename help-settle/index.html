🏪 店舗間ヘルプ 人件費精算 試作版 / たたき

店舗をまたいだ応援（ヘルプ）勤務を記録し、店舗間の人件費の貸し借りを自動で可視化・精算します。

💰 店舗別 人件費バランス

対象期間中、その店舗が「他店を応援して稼いだ額（貸し）」と「他店に応援してもらった額（借り）」の差引です。

プラス（貸し越し＝受け取る側） マイナス（借り越し＝支払う側）

🔁 店舗間 貸借マトリクス

縦＝応援を出した店（貸し手） / 横＝応援を受けた店（借り手）。金額は発生した人件費です。

🧾 精算リスト（相殺後）

店舗ペアごとに貸し借りを相殺した「最終的に誰が誰へいくら払うか」です。

＋ ヘルプ記録を追加

応援に入ったスタッフ・勤務先店舗・時間を入力すると、人件費（時間×時給）が自動計算されます。

📋 ヘルプ記録一覧

🏪 店舗マスタ

応援のやりとりをする店舗を登録します。

👤 スタッフマスタ

スタッフの所属店と基本時給を登録します（時給は記録時に上書き可）。

🗂 データ管理

この試作版はデータをブラウザ内（端末ローカル）に保存します。サーバー連携や複数人共有は本開発で対応予定です。

これは要件確認用の試作（たたき）です。実データの計算ロジック・権限・打刻連携などはヒヤリング後に詰めます。
