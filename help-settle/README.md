# 店舗間ヘルプ 人件費精算（試作版 / たたき）

> 多店舗運営で発生する「店舗間のヘルプ（応援勤務）」と、それによる人件費の貸し借りを
> 記録・可視化・精算するシングルファイルWebアプリの **試作（たたき）** です。
> 実店舗データは含まず、架空のサンプルデータで動作します。

🔗 **デモを開く / Live demo:** https://spicy-device.com/help-settle/

---

## これは何？
「ジョブカンの簡易版」イメージで、ヒヤリング前の叩き台として作成したものです。
スタッフが所属店以外の店舗を応援した実績を入力すると、人件費（時間×時給）を自動計算し、
店舗間の貸し借りをダッシュボードで可視化、相殺後の精算額まで算出します。

## 主な機能
- **ダッシュボード**: KPI（件数・総時間・移動総額）、店舗別 人件費バランス（棒グラフ）、
  店舗間 貸借マトリクス、相殺後の精算リスト（誰が誰へいくら払うか）
- **ヘルプ記録**: 日付・スタッフ・勤務先・時間を入力 → 人件費を自動計算。CSV出力・月次フィルタ
- **店舗・スタッフ管理**: 店舗／スタッフ（所属店・時給）の登録

## 技術 / 構成
- `index.html` 1ファイル完結（Vanilla JS、外部ライブラリ非依存）
- データ保存: ブラウザの localStorage（閲覧者の端末内のみ。他の人の入力は共有されません）

## このデモについて
- 起動時に架空のサンプルデータ（4店舗・8名・2か月分の応援実績）を自動投入します。
- 「店舗・スタッフ管理 → サンプルデータで初期化」でいつでも初期状態に戻せます。
- これは要件確認用の試作です。勤怠連携・割増賃金・複数人共有・承認フロー・認証などは本開発で対応予定です。
